# 📊 Olist E-Commerce Analytics Dashboard

## Project Overview

This project is an end-to-end E-Commerce Analytics Dashboard built using PostgreSQL and Power BI. The Olist dataset was loaded into PostgreSQL, cleaned, validated, and connected to Power BI using DirectQuery to create interactive business dashboards.

## Tools Used

* PostgreSQL
* SQL
* Power BI
* DAX
* DirectQuery

## Project Workflow

1. Loaded Olist dataset into PostgreSQL.
2. Performed data validation and cleaning.
3. Created indexes for query optimization.
4. Built SQL reporting views.
5. Connected Power BI to PostgreSQL using DirectQuery.
6. Created DAX measures and KPIs.
7. Developed interactive dashboards for business analysis.

## Dashboard Features

* Sales Analysis
* Customer Analysis
* Product Analysis
* Seller Analysis
* Review Analysis
* Logistics Analysis
